# 2. UML Class Diagram

## Node
- data: string
- *next: Node
- *previous: Node

## Deque
- size: int
- counter: int
- tmpData: string
- *head: Node
- *tail: Node
- *tmp: Node

- deque(int): void
- addFront(string): void
- addBack(string): void
- removeFront(): void
- removeBack(): void
- clear(): void
- checkSize(): void
- accessFront(): void
- accessBack(): void
- checkEmpty(): void
- find(string): void
- print(): void

Since deque class is the parent and the node is the child, a diamond shape is used from the Node class to the Deque class.
